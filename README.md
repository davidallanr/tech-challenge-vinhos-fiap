# Tech Challenge — Análise Estratégica do Setor Vitivinícola Brasileiro

Projeto desenvolvido para a Pós-Tech FIAP em Data Analytics.

## Objetivo

Analisar dados históricos do setor vitivinícola brasileiro entre 1970 e 2025, considerando exportações, importações, produção/processamento e fatores climáticos, com foco em geração de insights estratégicos para tomada de decisão.

## Fonte dos dados

Os dados foram obtidos na base pública Embrapa/Vitibrasil.

## Tecnologias utilizadas

- Python
- Google Colab
- Pandas
- Matplotlib
- PowerPoint

## Estrutura do projeto

- `notebook/`: notebook com tratamento, análise e geração dos gráficos.
- `dados/`: bases CSV utilizadas no projeto.
- `apresentacao/`: apresentação final em PDF e PowerPoint.

## Principais análises

- Maiores destinos das exportações brasileiras.
- Principais origens das importações.
- Tendência das exportações e importações.
- Transformação do perfil produtivo brasileiro.
- Impacto climático como variável estratégica.
- Recomendações de negócio para o setor.

## Conclusão

O setor vitivinícola brasileiro apresenta potencial competitivo relevante, mas o crescimento sustentável depende de diversificação de mercados, agregação de valor e gestão de riscos climáticos e comerciais.
